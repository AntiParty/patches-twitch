Command Documentation

Welcome to the FinalsRS documentation. This page explains how to customize and manage your Twitch bot commands easily

Configuration

Customizing Commands

You can fully customize the responses for !rank, !record, and !peak. Use variables in curly braces {variable} to inject dynamic player data into your custom messages.

How to edit

Use the !editcmd command directly in your Twitch chat:

// Example: Set a custom rank message
!editcmd rank @{username}, I am currently rank #{rank} with {rankScore} RS! 🔥
Variables

!rank Command

Show your current standing on the leaderboards.

{username} The Twitch user who called the command.
{rank} Current leaderboard rank (e.g. #450).
{league} Current league (e.g. Diamond 3).
{rankScore} Current Rank Score (RS) value.
{wtRank} Current World Tour rank.
Variables

!record Command

Track your session progress since you went live.

{sessionRS} RS gained or lost this session (e.g. +120).
{currentRS} Your total current RS.
{wtDiff} World Tour ranks gained/lost this session.
{startRS} Your RS at the start of the stream.
{wtRank} Your current World Tour rank.
Variables

!peak Command

Display your all-time highest achievements.

{rank} Highest rank ever achieved.
{league} League of peak rank (e.g. Diamond 1).
{rankScore} Peak Rank Score (RS) value.
{season} The season the peak was achieved.
{wtRank} Highest World Tour rank achieved.
{wtSeason} The WT season for that peak.
New System

Set Your Goals

The !goal command helps you track your journey to a specific rank.

Command Description
!goal <rank> Set a target rank (e.g. !goal 100)
!goal Display current goal progress
!goal remove Remove your current goal
