Bot Documentation

Comprehensive guide to managing and customizing your FinalsRS Twitch bot experience.

Essentials

Account Management

To start tracking your stats, you must link your THE FINALS account. Your Twitch account is linked automatically when you log in to the dashboard.

Command Description
!link <FinalsName#1234> Links your THE FINALS account to the bot.
!unlink Removes the link to your account.
Advanced

Customizing Commands

FinalsRS allows you to fully customize how the bot responds to commands like !rank, !record, and !peak. Variables wrapped in curly braces (e.g., {rank}) are replaced with real player data.

The !editcmd Command

Use !editcmd followed by the command name and your new message template.

// Set a custom rank message with flavor text
!editcmd rank Currently grinding at #{rank} with {rankScore} RS! Let's get it! 🔥

To reset a command to its default behavior, simply type !editcmd <command> without a message.

Reference

!rank Command

Displays current leaderboard rank, league standings, and World Tour progress.

{username} The Twitch user who called the command.
{rank} Current leaderboard rank (e.g. #450).
{league} Current league (e.g. Diamond 3).
{rankScore} Current Rank Score (RS) value.
{wtRank} Current World Tour rank.
Reference

!record Command

Tracks session progress (RS gained/lost) since you went live.

{sessionRS} RS gained or lost this session (e.g. +120).
{currentRS} Your total current RS.
{wtDiff} World Tour ranks gained/lost this session.
{startRS} Your RS at the start of the stream.
{wtRank} Your current World Tour rank.
Reference

!peak Command

Displays all-time highest achievements recorded by the system.

{rank} Highest rank ever achieved.
{league} League of peak rank (e.g. Diamond 1).
{rankScore} Peak Rank Score (RS) value.
{season} The season the peak was achieved.
{wtRank} Highest World Tour rank achieved.
{wtSeason} The WT season for that peak.
Information

!drops Command

Stay updated with active Twitch drops for THE FINALS.

!drops Lists current active drops and their durations.
!drop Alias for the drops command.
Utility

Rank Goals

The !goal system helps you track progress toward a specific leaderboard position.

Command Description
!goal <rank> Set a target rank (e.g. !goal 100)
!goal Display current goal progress & percentage.
!goal remove Clear your current active goal.
Premium / Tester

Predict Cutoff

The !predict command uses historical T500 trends to forecast future leaderboard cutoffs.

// Forecast the cutoff in 30 days
!predict 30
Utility

Utility Commands

Standard commands for interacting with the bot and discovering available features.

Command Description
!commands Lists all available commands for the channel.
!help [command] Get detailed usage info for a specific command.
!ping Check if the bot is responsive and view latency.
