Customize Your Twitch Bot Commands

Learn how to personalize your FinalsRR bot responses for commands like !rank and !record with custom messages and variables.

1View Current Command Settings

To see what your bot currently says for a command, use:

!editcmd rank

This will display the current custom message (if set) or show the default response for the !rank command in your channel.

2Customize Command Responses

Change what your bot says when viewers use specific commands:

!editcmd rank FinalsRR: @{username}, #{rank} {league} - {rankScore} RS | WT rank: #{wtRank}

After setting this, when someone uses !rank, your bot will respond with your custom message, automatically filling in the user-specific details.

Available Variables

Use these special variables in your custom commands (wrap them in curly braces):

Variable Description Example Output
{username} The viewer's Twitch username CoolStreamer92
{rank} Their current rank 1250
{league} Their league ranking Diamond
{rankScore} Their rank score 3450
{wtRank} Their World Tour rank 75
{found} Returns 'true' if player found, 'false' if not true
{sessionRS} Session rank score (for !record command) +150
{currentRS} Current rank score (for !record command) 3600
Note: Available variables depend on the specific command. Not all variables work with every command.

3Reset to Default Message

To remove your custom message and return to the bot's default response:

!editcmd rank

This will clear any custom message set for the specified command.

Customizing Other Commands

You can customize other supported commands using the same format. For example, with !record:

!editcmd record @{username}, your session RS is {sessionRS} ({currentRS} RS)

Practical Examples

Example 1: Basic Rank Command

Command: !editcmd rank {username} is rank #{rank} in {league} with {rankScore} RS

Result: "CoolStreamer92 is rank #1250 in Diamond with 3450 RS"

Example 2: Record Command with Emotes

Command: !editcmd record PogChamp @{username}'s session: {sessionRS} | Total: {currentRS} RS

Result: "PogChamp @CoolStreamer92's session: +150 | Total: 3600 RS"

Example 3: Player Not Found Handling

Command: !editcmd rank Sorry {username}, player data not found {NotLikeThis}

Result when player not found: "Sorry CoolStreamer92, player data not found NotLikeThis"

Tips & Best Practices

  • Only channel owners and moderators should use !editcmd for security reasons
  • Test your commands with different variables to ensure they work as expected
  • Keep messages concise to avoid spam in your chat
  • You can include Twitch emotes in your custom commands for more personality
  • If a command isn't working, check that you're using variables available for that specific command
Warning: Avoid using overly long custom messages as they may be truncated or appear as spam to your viewers.
