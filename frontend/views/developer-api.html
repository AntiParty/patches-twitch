Developer API

Access our calculated datasets to build amazing community tools for THE FINALS. Public endpoints, no API key required.

Getting Started

Rate Limits & CORS

The API is currently public and does not require an API key. To ensure fair usage, we implement IP-based rate limiting.

  • Base URL: https://finalsrs.com/api/v1
  • Rate Limit: 60 requests per minute per IP
  • CORS: Enabled for all endpoints
Endpoint

Leaderboard Prediction

Returns the estimated Top 500 cutoff Rank Score (RS) for a future date, accounting for daily trends and end-of-season rushes.

GET /leaderboard/cutoff

Query Parameters

Parameter Type Description
days Integer Optional. Number of days into the future to predict. Defaults to current season end.

Response

{
  "meta": {
    "generated_at": "2026-01-23T12:00:00.000Z",
    "api_version": "v1"
  },
  "data": {
    "current_cutoff_rs": 45100,
    "prediction": {
      "target_date_days": 55,
      "predicted_rs": 62500,
      "confidence_interval": {
        "min": 61200,
        "max": 63800
      },
      "trend": {
        "daily_change": 320,
        "slope_standard_error": 15
      }
    },
    "confidence_level": "High"
  }
}
Endpoint

Player Stats

Retrieve real-time leaderboard statistics for a specific player. Supports searching by exact name or name#tag.

GET /player/:name

Path Parameters

Parameter Type Description
name String The Embark ID (Name#1234) or display name to search for.
Important: Player names with tags require URL encoding. Replace # with %23
Example: https://finalsrs.com/api/v1/player/carnifex%237330

Response

{
  "meta": {
    "generated_at": "2026-01-23T12:05:00.000Z"
  },
  "data": {
    "name": "EmbarkName#1234",
    "rank": 42,
    "league": "Diamond 1",
    "rank_score": 58200,
    "movement": {
      "rank_change": -2,
      "score_change": 150
    },
    "updated_at": "2026-01-23T11:45:00.000Z"
  }
}
