FinalsRS Bot Documentation

Welcome to the FinalsRS documentation. This page explains how to customize and manage your Twitch bot commands easily.

View Your Current Command

To check your bot’s current message for a command:

!editcmd rank

Customize a Command

Change what your bot says when a viewer uses a command:

!editcmd rank @{username}, #{rank} {league} - {rankScore} RS | WT rank: #{wtRank}

Available Variables

VariableDescriptionExample
{username}The viewer's Twitch usernameCoolStreamer92
{rank}Their current rank#948
{league}League tierDiamond 3
{rankScore}Rank score43,331 RS
{wtRank}World Tour rank#470

Reset a Command

To reset a command to its default message:

!editcmd rank

Example

This example customizes the !record command:

!editcmd record @{username}'s session: {sessionRS} | Total: {currentRS} RS
