Why This Was Developed

As a huge fan and streamer of THE FINALS, I’ve really wanted the developers to provide more official APIs. With better access to game data, creators like me could build tools that make streams more engaging — real-time player stats, live leaderboards, interactive Twitch commands, and beyond.

Right now, the ranked leaderboard API is a great start, but it only scratches the surface. That’s why I built FinalsRR — to bring that leaderboard data together with live stats and rank tracking, creating a Twitch bot that streamers and viewers can rely on.

I’m excited about what’s possible if the devs expand their API offerings. More open APIs would empower the community to create innovative projects that elevate the whole THE FINALS experience. I hope this encourages the developers to continue supporting creators by unlocking even more data.
