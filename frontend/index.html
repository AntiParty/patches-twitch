Frequently Asked Questions

Do I need to be a Twitch Affiliate to use FinalsRR?

Nope! Anyone can use FinalsRR — whether you’re a beginner or partnered streamer.

Does the bot need moderator access?

Yes — the bot needs to be a moderator in your Twitch chat to respond to commands like !rank or !kd. Without mod access, it won’t be able to reply in chat.

How often does the rank update?

Updates are fetched every 40-45 minutes. Leaderboard is cached then stored for use

Pricing

FinalsRR is currently completely free to use. In the future, we may introduce optional premium tools to help support the bot's upkeep and continued development — but the core features will always remain free.

Why This Was Developed

As a huge fan and streamer of THE FINALS, I’ve really wanted the developers to provide more official APIs. With better access to game data, creators like me could build tools that make streams more engaging — real-time player stats, live leaderboards, interactive Twitch commands, and beyond. scratches the surface. That’s why I built FinalsRR — to bring that leaderboard data together with live stats and rank tracking, creating a Twitch bot that streamers and viewers can rely on.

I’m excited about what’s possible if the devs expand their API offerings. More open APIs would empower the community to create innovative projects that elevate the whole THE FINALS experience. I hope this encourages the developers to continue supporting creators by unlocking even more data.
